# :sparkles: appicons

A collection of application logos for direct use in web and project.

# :file_folder: Structure

```bash
appicons/
├── svg/    # Vector icons (preferred)
└── png/    # Raster icons (fallback)
```

# :rocket: Usage

## CDN (Recommended)

```html
<img src="https://cdn.jsdelivr.net/gh/kyranhsiao/appicons/svg/{app_name_here}.svg" width="48">
```

## Raw GitHub URL

```html
<img src="https://raw.githubusercontent.com/kyranhsiao/appicons/main/svg/{app_name_here}.svg">
```
